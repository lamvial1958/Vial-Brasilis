# A1.2 — Família e Amigos

## 1. Diálogo de Abertura

**Cena — Tom mostra uma foto da família para Li:**
**Tom:** - Olha, essa é a minha família.

**Li:** - Que legal! Quem é essa mulher?

**Tom:** - Essa é a minha mãe. Ela se chama Helen.

**Li:** - E o homem ao lado dela?

**Tom:** - É o meu pai, Richard. E essa é a minha irmã, Emma.

**Li:** - Ela é muito parecida com você!

**Tom:** - É verdade! Ela é alta e simpática, como eu.

**Li:** - E quem são essas crianças?

**Tom:** - São os filhos da minha irmã — meus sobrinhos.


**Quadro funcional:**
| Função | Fórmula |
|---|---|
| Apresentar uma pessoa da família | Esse é o meu... / Essa é a minha... |
| Perguntar quem é alguém | Quem é essa pessoa? / Quem é esse/essa? |
| Descrever aparência/personalidade | Ele é alto/baixo/simpático... |
| Indicar semelhança | Ele/ela é parecido(a) com... |

## 2. Vocabulário

pai, mãe, irmão, irmã, filho, filha, avô, avó, tio, tia, primo, prima, sobrinho, sobrinha, marido, esposa, namorado, namorada, amigo, amiga, alto/alta, baixo/baixa, simpático/simpática, parecido/parecida

## 3. Gramática em Foco

**A. Possessivos** — concordam com o substantivo possuído, não com quem possui:
| | Singular | Plural |
|---|---|---|
| (eu) | meu pai / minha mãe | meus pais / minhas tias |
| (você/ele/ela) | seu pai / sua mãe | seus pais / suas tias |

**B. Adjetivos descritivos** — concordância de gênero e número:
alto/alta/altos/altas · baixo/baixa/baixos/baixas · simpático/simpática/simpáticos/simpáticas

**C. "Parecido(a) com"** — para indicar semelhança: *Ela é parecida com a mãe. Eles são parecidos com o pai.*

## 4. Bloco de Pronúncia

**A. "lh" /ʎ/ (revisão):** filho, filha, mulher (atenção: "lh" ≠ "l" — não confundir com "filo")
**B. Nasalização em parentesco:** mãe, irmã, também
**C. Acento tônico em "parecido":** pa-re-**ci**-do

## 5. Exercícios (com gabarito)

**Exercício 1 — Complete o diálogo:**
1. Essa é a ___ mãe. → **minha**
2. Quem é ___ mulher? → **essa**
3. Ela é muito ___ com você. → **parecida**
4. São os filhos da minha irmã, meus ___. → **sobrinhos**

**Exercício 2 — Complete com o possessivo correto:**
"Esse é (eu) ___ pai" → **meu** · "Essas são (você) ___ tias" → **suas** · "Esses são (eu) ___ amigos" → **meus**

**Exercício 3 — Concorde o adjetivo:**
Ela é alt___ (**alta**) · Eles são simpátic___ (**simpáticos**) · Elas são baix___ (**baixas**)

**Exercício 4 — Associe o parentesco à definição:**
filho do meu irmão → **sobrinho** · mãe do meu pai → **avó** · irmão da minha mãe → **tio**

## 6. Listening — Script e Perguntas

**Marco:** - Essa é a minha família. Esse é o meu pai, Giuseppe, e essa é a minha mãe, Anna. Eu tenho dois irmãos: o Paolo, que é alto e muito simpático, e a Giulia, que é baixa.

**Entrevistador:** - Você é parecido com alguém da família?

**Marco:** - Sim, eu sou parecido com o meu pai.


**Perguntas:**
1. Como se chamam os pais de Marco? → **Giuseppe e Anna**
2. Quantos irmãos Marco tem? → **Dois**
3. Com quem Marco é parecido? → **Com o pai**

## 7. Tarefa de Produção Final

Descreva (oral ou por escrito) 3 pessoas próximas a você — uma da família e duas amigas — usando: quem é (relação), como se chama, como é (aparência/personalidade) e, se possível, uma semelhança com outra pessoa.

**Checklist de autoavaliação:**
- [ ] Usei "Esse é o meu.../Essa é a minha..." para apresentar cada pessoa
- [ ] Usei pelo menos 2 possessivos diferentes corretamente
- [ ] Concordei os adjetivos descritivos em gênero e número
- [ ] Usei "parecido(a) com" pelo menos uma vez
- [ ] Pronunciei corretamente uma palavra com "lh"

## 8. Glossário da Unidade

| Palavra/Expressão | Significado |
|---|---|
| alto/alta | de estatura grande |
| amigo/amiga | pessoa querida que não é da família |
| avô/avó | pai ou mãe do pai/da mãe |
| baixo/baixa | de estatura pequena |
| esposa / marido | cônjuge da pessoa |
| esse é o meu... / essa é a minha... | fórmula para apresentar uma pessoa |
| filho/filha | descendente direto de alguém |
| irmão/irmã | filho(a) dos mesmos pais |
| namorado/namorada | parceiro(a) romântico(a), sem casamento |
| pai/mãe | progenitores |
| parecido/parecida (com) | semelhante a alguém |
| primo/prima | filho(a) de um tio/tia |
| quem é essa pessoa? | pergunta para identificar alguém |
| simpático/simpática | de personalidade agradável |
| sobrinho/sobrinha | filho(a) de um irmão/irmã |
| tio/tia | irmão/irmã de um dos pais |
