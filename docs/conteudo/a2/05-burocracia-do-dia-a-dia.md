# A2.5 — Burocracia do Dia a Dia

*Ficha pedagógica completa em `docs/unidades/a2.md`. Primeira aproximação à linguagem formal, indispensável para resolver situações práticas em bancos, correios e repartições no Brasil.*

## 1. Diálogo de Abertura

**Cena — atendimento numa agência bancária:**
**Atendente:** - Bom dia! Em que posso ajudar?

**Cliente:** - Bom dia. Eu gostaria de abrir uma conta corrente.

**Atendente:** - Sem problema. Você tem CPF e comprovante de residência?

**Cliente:** - Tenho sim, aqui estão os documentos.

**Atendente:** - Perfeito. Seria possível preencher este formulário, por favor?

**Cliente:** - Claro. É só assinar no final?

**Atendente:** - Sim, e depois eu preciso de uma foto sua para o cadastro. Aqui está o seu protocolo de atendimento.

**Cliente:** - Obrigado. Quanto tempo demora para a conta ficar pronta?

**Atendente:** - Em até cinco dias úteis, você recebe o cartão em casa.


**Quadro funcional:**
| Função | Fórmula |
|---|---|
| Pedir algo formalmente | Eu gostaria de... / Seria possível...? |
| Oferecer ajuda formalmente | Em que posso ajudar? |
| Confirmar disponibilidade de documento | Você tem...? / Tenho sim. |
| Pedir para preencher algo | Seria possível preencher...? |
| Perguntar prazo | Quanto tempo demora para...? |
| Indicar prazo | Em até X dias (úteis) |

## 2. Vocabulário

| Termo | Definição simples | Exemplo |
|---|---|---|
| CPF | número de identificação fiscal no Brasil | Preciso do meu CPF para abrir a conta. |
| comprovante de residência | documento que prova onde você mora | Trouxe meu comprovante de residência. |
| formulário | documento para preencher dados | Preencha este formulário. |
| protocolo | número que identifica um atendimento | Guarde o número do protocolo. |
| senha de atendimento | número/código para ser chamado | Pegue sua senha de atendimento. |
| conta corrente | tipo de conta bancária para o dia a dia | Quero abrir uma conta corrente. |
| cadastro | registro de dados de uma pessoa | Preciso fazer o cadastro. |
| assinar | escrever o próprio nome num documento | Assine aqui, por favor. |
| dias úteis | dias de funcionamento normal (sem fins de semana/feriados) | Demora cinco dias úteis. |
| repartição (pública) | órgão público de atendimento | Fui à repartição renovar o documento. |
| encomenda | pacote/correspondência | Vou retirar uma encomenda no correio. |

## 3. Gramática em Foco

**A. Linguagem formal de pedido:**
| Estrutura | Exemplo | Nível de formalidade |
|---|---|---|
| Eu quero + verbo | Eu quero abrir uma conta. | informal |
| Eu gostaria de + verbo | Eu gostaria de abrir uma conta. | formal/educado |
| Seria possível + verbo? | Seria possível preencher este formulário? | mais formal |

**B. "Gostaria" é o condicional de "gostar"** — usado de forma fixa para pedidos educados, sem precisar conjugar outras formas do condicional neste nível.
**C. Vocabulário de documentos:** combine com substantivos formais (comprovante, formulário, protocolo) em vez de equivalentes informais.

## 4. Bloco de Pronúncia

**A. "Gostaria" — ditongo "ia" final, átono e rápido:** gosta-RI-a, sem acento na última sílaba.
**B. Sigla "CPF" pronunciada letra por letra:** "cê-pê-éfe", nunca como uma palavra só.
**C. "Seria possível" — entonação ascendente de pergunta educada,** mais suave que uma pergunta direta.

## 5. Exercícios (com gabarito)

**Exercício 1 — Transforme em pedido formal:**
"Eu quero abrir uma conta" → **Eu gostaria de abrir uma conta.**
"Você pode preencher isso?" → **Seria possível preencher isso?**

**Exercício 2 — Complete o diálogo de atendimento:**
1. ___ posso ajudar? → **Em que**
2. Eu ___ de retirar uma encomenda. → **gostaria**
3. ___ possível ver meu protocolo? → **Seria**

**Exercício 3 — Associe o documento à situação:**
(CPF / comprovante de residência / formulário) com (abrir conta / provar endereço / preencher dados) → **CPF → identificação fiscal; comprovante de residência → provar endereço; formulário → preencher dados**

**Exercício 4 — Complete com vocabulário de burocracia:**
Depois de preencher o ___, guarde o número do ___ para acompanhar seu atendimento. → **formulário, protocolo**

## 6. Compreensão Oral — Transcrição e Perguntas

**Atendente dos Correios:** - Bom dia! Em que posso ajudar?

**Cliente:** - Bom dia. Eu gostaria de retirar uma encomenda.

**Atendente:** - Você tem o código de rastreamento e um documento com foto?

**Cliente:** - Tenho sim, aqui está.

**Atendente:** - Seria possível assinar aqui para confirmar o recebimento?

**Cliente:** - Claro. Pronto.

**Atendente:** - Perfeito, aqui está sua encomenda. Tenha um bom dia!


**Perguntas:**
1. O que o cliente quer fazer? → **Retirar uma encomenda**
2. O que ele precisa apresentar? → **Código de rastreamento e documento com foto**
3. O que ele precisa fazer para confirmar o recebimento? → **Assinar**

## 7. Tarefa de Produção Final

Faça um role-play (em par, ou grave os dois papéis) simulando um atendimento bancário ou nos correios. Inclua:
1. Pelo menos 2 pedidos formais com "eu gostaria de" ou "seria possível"
2. Pelo menos 3 itens de vocabulário de documentos (CPF, comprovante, formulário, protocolo...)
3. Uma pergunta sobre prazo e a resposta correspondente

**Checklist de autoavaliação:**
- [ ] Usei "eu gostaria de" para pedidos formais
- [ ] Usei "seria possível...?" para pedidos mais formais ainda
- [ ] Usei vocabulário correto de documentos brasileiros
- [ ] Perguntei e respondi sobre prazo (dias úteis)
- [ ] Pronunciei a sigla CPF letra por letra
