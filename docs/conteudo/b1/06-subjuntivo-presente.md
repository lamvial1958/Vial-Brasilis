# B1.6 — Subjuntivo Presente

*Ficha pedagógica completa em `docs/unidades/b1.md`. Primeiro item gramatical estrutural novo do nível B1: subjuntivo presente após verbos de desejo/dúvida/recomendação — pré-requisito direto para B1.8 e B2.1.*

## 1. Diálogo de Abertura

**Cena — conselhos a um colega recém-chegado ao Brasil:**

**Marco:** - Acabei de chegar, ainda tô meio perdido aqui. Você tem algum conselho?

**Renata:** - Tenho! Primeiro, é importante que você abra uma conta bancária o quanto antes — muita coisa depende disso.

**Marco:** - Faz sentido. E sobre morar, você recomenda algum bairro?

**Renata:** - Recomendo que você visite alguns bairros antes de decidir, porque o custo de vida varia muito. Espero que você encontre um lugar que combine com seu orçamento.

**Marco:** - E sobre o idioma? Acha que eu deveria fazer um curso?

**Renata:** - Sugiro que você pratique todos os dias, mesmo que seja pouco tempo. Talvez seja difícil no início, mas duvido que você demore muito para se adaptar.


**Quadro funcional:**
| Função | Fórmula |
|---|---|
| Expressar desejo/esperança | Espero que você encontre... |
| Expressar recomendação | Recomendo/Sugiro que você visite... |
| Expressar necessidade impessoal | É importante/preciso que você abra... |
| Expressar dúvida | Duvido que você demore muito. |
| Expressar possibilidade | Talvez seja difícil no início. |

## 2. Vocabulário

| Termo | Definição simples | Exemplo |
|---|---|---|
| espero que | introduz desejo sobre algo incerto | Espero que você encontre um bom lugar. |
| recomendo que | introduz uma recomendação | Recomendo que você visite os bairros. |
| sugiro que | introduz uma sugestão | Sugiro que você pratique todo dia. |
| é importante que | introduz uma necessidade impessoal | É importante que você abra uma conta. |
| é preciso que | introduz uma necessidade forte | É preciso que você tenha paciência. |
| talvez | introduz possibilidade incerta | Talvez seja difícil no início. |
| duvido que | introduz dúvida/descrença | Duvido que você demore muito. |
| quero que | introduz desejo direcionado a outra pessoa | Quero que você se sinta bem-vindo. |
| é bom que | introduz avaliação positiva sobre algo incerto/futuro | É bom que você já tenha amigos aqui. |
| espero conseguir | expressão de esperança sobre si mesmo | Espero conseguir me adaptar rápido. |
| combinar com (algo) | ser adequado/compatível com | Espero que combine com seu orçamento. |
| custo de vida | quantia necessária para viver num lugar | O custo de vida varia muito por bairro. |
| pé no chão (figurado) | atitude realista/sem expectativas exageradas | É bom manter o pé no chão no início. |
| se adaptar | se ajustar a uma nova situação | Duvido que demore para se adaptar. |

## 3. Gramática em Foco

**A. Formação do subjuntivo presente — verbos regulares:**
| Infinitivo | Indicativo (eu) | Subjuntivo (que eu/você/ele) |
|---|---|---|
| falar (-ar) | falo | que eu fale / que você fale |
| comer (-er) | como | que eu coma / que você coma |
| abrir (-ir) | abro | que eu abra / que você abra |

> **Regra prática:** parte da forma "eu" do presente do indicativo, troca a vogal temática (-ar→-e / -er,-ir→-a).

**B. Verbos irregulares frequentes no subjuntivo presente:**
| Verbo | que eu | que você/ele | que nós |
|---|---|---|---|
| ser | seja | seja | sejamos |
| estar | esteja | esteja | estejamos |
| ter | tenha | tenha | tenhamos |
| ir | vá | vá | vamos |
| fazer | faça | faça | façamos |

**C. Gatilhos que exigem subjuntivo (verbos/expressões da oração principal):**
| Categoria | Gatilhos | Exemplo |
|---|---|---|
| Desejo | espero que, quero que | Espero que você consiga. |
| Recomendação | recomendo que, sugiro que | Sugiro que você pratique. |
| Necessidade impessoal | é importante que, é preciso que | É preciso que você se organize. |
| Dúvida | duvido que, não acho que | Duvido que seja fácil. |
| Possibilidade | talvez | Talvez seja difícil. |

> **Nota:** quando não há incerteza/desejo/dúvida — apenas afirmação de fato — usa-se o indicativo: "Sei que ele **vem**" (fato, indicativo) vs. "Espero que ele **venha**" (desejo, subjuntivo).

## 4. Bloco de Pronúncia

**A. Contraste indicativo × subjuntivo no mesmo verbo:** "ele **fala**" (indicativo) × "que ele **fale**" (subjuntivo) — vogal final muda de "a" para "e", atenção à abertura vocálica.

**B. "Talvez" — ditongo "al" fechado:** tal-VEZ, sem abrir demais a vogal "a".

## 5. Exercícios (com gabarito)

**Exercício 1 — Complete com o subjuntivo presente:**
1. Espero que você ___ (conseguir) se adaptar rápido. → **consiga**
2. É importante que nós ___ (ter) paciência. → **tenhamos**
3. Recomendo que ele ___ (visitar) outros bairros. → **visite**

**Exercício 2 — Indicativo ou subjuntivo? Escolha a forma correta:**
1. Eu sei que ele ___ (morar) no centro. → **mora** (fato, indicativo)
2. Espero que ele ___ (morar) perto do trabalho. → **more** (desejo, subjuntivo)

**Exercício 3 — Complete com o verbo irregular adequado no subjuntivo:**
"Talvez ___ (ser) difícil no início, mas espero que tudo ___ (dar) certo." → **seja / dê**

**Exercício 4 — Transforme a recomendação direta em recomendação com subjuntivo:**
"Visite outros bairros antes de decidir." → **Recomendo que você visite outros bairros antes de decidir.**

## 6. Compreensão Oral — Transcrição e Perguntas

> **Apresentador (programa de conselhos para estrangeiros):**
> Se você está chegando ao Brasil agora, aqui vão algumas dicas. É importante que você organize a documentação nos primeiros dias. Recomendo que você não tenha pressa para tomar decisões grandes, como assinar um contrato de aluguel. Espero que você encontre uma comunidade de apoio, seja de outros estrangeiros, seja de brasileiros. E talvez seja difícil no início, mas duvido que você se arrependa de ter vindo.

**Perguntas:**
1. Que recomendação é dada sobre documentação? → **Organizá-la nos primeiros dias**
2. Que recomendação é dada sobre decisões grandes? → **Não ter pressa, como ao assinar contrato de aluguel**
3. O que o apresentador espera para o ouvinte? → **Que encontre uma comunidade de apoio**
4. Qual é a opinião final do apresentador, expressa com "duvido que"? → **Que o ouvinte não vai se arrepender de ter vindo**

## 7. Tarefa de Produção Final

Escreva (ou grave) uma lista de conselhos para um colega recém-chegado ao Brasil (oral ou escrito, 150 palavras / 1–2 minutos), com justificativa para cada conselho. Inclua:
1. Pelo menos 3 conselhos usando gatilhos diferentes de subjuntivo (espero que, recomendo que, é importante que, sugiro que)
2. Pelo menos 1 verbo irregular no subjuntivo (ser, estar, ter, ir, fazer)
3. Uma frase com "talvez" expressando possibilidade
4. Uma justificativa para cada conselho (por quê)

**Checklist de autoavaliação:**
- [ ] Usei pelo menos 3 gatilhos diferentes de subjuntivo corretamente
- [ ] Usei pelo menos 1 verbo irregular no subjuntivo
- [ ] Distingui corretamente quando usar indicativo (fato) e subjuntivo (desejo/dúvida)
- [ ] Justifiquei cada conselho dado
